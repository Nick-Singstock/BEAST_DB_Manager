This is the main control file for using gc_manager.py in a managed directory.
Please see the readme.txt file if you are unfamiliar with the manager script.	

Useful Syntax: 
'#' for comments
'=SURF' for surface
'-MOL' for adsorbate molecules
'Biases: [biases]' for listing biases
'Dist: int' for setting adsorbate distance
'Desorb: [biases]' for desorbed molecule biases
'+tag' for adding inputs tags
'+[#] tag' for adding tags to convergence file at step #
'NEB: init/folder final/folder nimages path_name [biases]' for adding NEB pathways
